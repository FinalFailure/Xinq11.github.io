linux和vim

一. Linux

Linux是一个操作系统,是硬件与其他软件沟通的桥梁

1. 命令格式

命令[-选项] [参数]

例:ls -la/etc

  • 分别命令使用不遵循此格式
  • 当有多个选项时可以写在一起
  • 简化选项与完整选项 -a 简化 -all 完整

    2. 常用命令

    1. 目录处理命令
    1.ls：
  • 功能:显示目录文件
  • 语法:ls 选项[-ald] [文件或目录]
    - -a: 显示所有文件,包括隐藏文件（linux下隐藏文件名以.开头）
- -l: 显示详细信息 （-lh:更好显示文件大小）
- -d: 查看目录属性
- -i: 查看文件i结点
    2.mkdir：
  • 功能:创建新目录
  • 语法:mkdir -p[目录名] 递归创建 mkdir [目录名]
    • 如果一个目录本身不存在,可以用-p在这个本身不存在目录创建的同时再在这个目录下面创建一个子目录
    • Linux下用mkdir可以同时创建多个目录，不同于windows一次只能创建一个
      3.cd：
  • 功能:切换目录
  • 语法:cd [目录] cd.. :返回目前目录位置的上一层目录.
    4.pwd：
  • 功能:显示当前目录的绝对路径
    • 绝对路径：由根目录写起。例如/home/tuyx/test/file1
    • 相对路径：不是从/写起，而是指相对当前工作目录的路径，
      例如由/home/tuyx/test/file1 要到/home/tuyx/test/file2 下面时，可以写成 “cd ../file2”.
      5.rmdir:
  • 功能:删除空目录
  • 语法:rmdir [目录名]
    6.cp：
  • 功能:复制文件或目录
  • 语法: cp -rp [原文件或目录] [目标目录]
    - -r: 复制目录
- -p: 保留文件属性
    7.mv：
  • 功能:剪切文件,改名 可一次移动多个文件
  • 语法: mv [原文件或目录] [目标目录]
    8.rm：
  • 功能:删除文件或目录
  • 语法: rm -rf [文件或目录]
    - -r:删除目录
- -f:强制执行
- -rf:强制删除目录
    2.文件处理命令
    1.touch
  • 功能:创建空文件
  • 语法: touch [文件名]
    2.cat
  • 功能:显示文件内容（适合短文件）
  • 语法:cat [文件名] -n 显示行号
    3.tac
  • 功能:显示文件内容（反向列式）
  • 语法:tac [文件名]
    4.more
  • 功能:分页显示文件内容（适合显示长文件,但不可向上翻页）
  • 语法:more [文件名]
    - （空格）或f  翻页
- （Enter）   换行
- q或Q        退出
    5.less
  • 功能:分页显示文件内容（可向上翻页）
  • 语法:less [文件名]
    - pageup      向上翻页
-  上箭头      向上一行
- /关键词     可搜索（n可查看匹配的关键词）
-  其余与more相同
    6.head
  • 功能:显示文件前面几行
  • 语法:head [文件名]
    - -n 数字:指定显示行数（默认十行）
    7.tail
  • 功能:显示文件后面几行
  • 语法:tail [文件名]
    - -n:指定行数（默认十行）
- -f:动态显示文件末尾内容（做变动时可动态显示）
    8.ln
  • 功能:生成链接文件
  • 语法: ln -s [原文件] [目标文件] 创建软链接 
    - ln [原文件] [目标文件] 创建硬链接,不能针对目录使用
    （软链接类似windows的快捷方式,硬链接类似cp -p（拷贝），但硬链接可同步更新,如果删除原文件,软链接会报错原文件丢失,硬链接依然可以访问）
    3.权限管理命令
    1.chmod
  • 功能:改变目录或文件权限
  • 语法: chmod [{ugoa} {+-=} {rwx}] [文件或目录]
    - [mode=421] [文件或目录]
- -R 递归修改 修改目录及其目录下文件的权限
- u:所有者 g:所属组 o:其他人 a:所有人
- +:增加权限 -:减少权限 =:直接赋予权限,不管之前是什么权限
- r:读 代表数字 4  w:写 数字2  x:执行 数字 1
    2.chown
  • 功能:改变文件或目录的所有者
  • 语法:chown [用户] [文件或目录]
    • 在linux中只有root管理者用户可以改变一个文件或目录的所有者
3.chgrp
  • 功能:改变文件或目录的所属组
  • 语法: chgrp [用户组] [文件或目录]
    4.umask
  • 功能:显示,设置文件的缺省权限
  • 语法:umask [-S]
    - -S 以rwx形式显示新建文件缺省条件
- umask [mode=421] 重新设置文件的缺省权限,在linux下用新建文件默认是没有执行权限
    4.文件搜索命令
    1.find
    • 功能:文件搜索
    • 语法:find [搜索范围] [匹配条件]
      • find /etc -name init在目录/etc中查找init文件 -iname不区分大小写
      • find /-size +204800在根目录下查找大于100MB的文件,+n大于,-n小于,n等于
      • find /home -user xxx,在根目录下查找所有者为xxx的文件,-group根据所属组查找
      • 如果搜索条件有两个
      • -a:满足两个条件
      • -o:满足任一条件
      • find /etc -name inittab -exec ls -l {};在/etc下查找inittab文件并显示其详细信息 -exec/-ok（多一个询问确认的环节） 命令{}; 对搜索结果执行操作
      • -type 根据文件类型查找 f文件 d目录 l软链接文件,-inum根据i结点查找
2.locate
  • 功能:在文件资料库中查找文件,相对find查找速度会更快
  • 语法: locate 文件名,-i 不区分大小写,updatedb 手动更新文件库
3.which
  • 功能:所搜命令所在目录及别名信息
  • 语法: which 命令 例: which ls
4.grep
  • 功能:在文件中搜索字符串匹配的行并输出
  • 语法: grep -iv [指定子串] [文件]
    • -i 不区分大小写
    • -v 排除指定字串
5.帮助命令
1.man
  • 功能:获取帮助信息
  • 语法: man [命令或配置文件]
6.用户管理命令
1.useradd
  • 功能:添加用户
  • 语法: useradd 用户名
2.passwd
  • 功能:设置用户密码
  • 语法: passwd 用户名
    3.who
  • 功能:查看登录用户信息
    7.压缩解压命令
    1.gzip
  • 功能:压缩文件（只能压缩文件,不能压缩目录,且不保留原文件）
  • 语法: gzip [文件]
    • 压缩后文件格式 .gz
      2.gunzip
  • 功能:解压缩.gz的压缩文件
  • 语法: gunzip [压缩文件]
    3.tar
  • 功能:打包目录
  • 语法: tar 选项[-zcf] [压缩后文件名] [目录] 压缩
    • tar 选项[-zxf] [压缩后文件名] 解压
    • -c 打包
    • -v 显示详细信息
    • -f 指定文件名
    • -z 打包同时压缩
    • -x 解包
    • -f 指定解压文件
    • -z 解压缩
    • 压缩后文件格式: .tar.gz
      4.zip
  • 功能:压缩文件或目录（压缩后保留源文件）
  • 语法: zip 选项[-r] [压缩后文件名] [文件或目录]
    • -r 压缩目录
    • 压缩后文件格式 .zip
      5.unzip
  • 功能:解压缩.zip的文件
  • 语法: unzip [压缩文件]
